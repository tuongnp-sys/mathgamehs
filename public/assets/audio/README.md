# Background music (MP3)

Place these files here (game runs silently if missing):

| File | When it plays |
|------|----------------|
| `hub-upbeat.mp3` | After opening the treasure envelope on the world hub (loops until the quest starts) |
| `result-calm.mp3` | After submitting an exam on the result / treasure screen (loops until leaving that screen) |

Edit files in this folder only — not `dist/` — then rebuild or refresh dev server.
